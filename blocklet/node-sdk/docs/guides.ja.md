# ガイド

Blocklet SDK ガイドへようこそ。このセクションでは、Webアプリケーションにおける一般的な課題を解決し、主要な機能を実装するのに役立つ、実践的なステップバイステップのチュートリアルを提供します。

アプリケーションの異なる部分間で安全な通信を可能にしたり、強力なビルド済みミドルウェアでサーバーを強化したりする場合でも、これらのガイドは迅速に立ち上げられるように設計されています。

<x-cards data-columns="2">
  <x-card data-title="コンポーネント間通信" data-icon="lucide:arrow-right-left" data-href="/guides/component-communication">
    `component.call` メソッドを使用して、blocklet内の異なるコンポーネント間で安全な通信を有効にします。
  </x-card>
  <x-card data-title="Webサーバーミドルウェアの使用" data-icon="lucide:shield" data-href="/guides/using-middlewares">
    CSRF保護、サイトマップ生成、SPAフォールバックのためのすぐに使えるミドルウェアで、Express.jsサーバーを強化します。
  </x-card>
</x-cards>

これらのガイドに従うことで、より複雑で堅牢なBlockletアプリケーションを構築するための強固な基盤を築くことができます。利用可能なすべての関数とモジュールを包括的に理解するためには、[APIリファレンス](./api-reference.md)を詳しくご覧になることをお勧めします。
