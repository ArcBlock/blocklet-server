# API 參考

歡迎來到 `@blocklet/js-sdk` 的 API 參考文件。本節為 SDK 提供的所有類別、方法、函式和型別提供詳細的技術文件。它旨在幫助需要精確理解 SDK 元件及其功能的開發人員。

無論您是想建立自訂 API 客戶端、與特定的 Blocklet 服務互動，還是理解資料結構，都可以在這裡找到必要的詳細資訊。
