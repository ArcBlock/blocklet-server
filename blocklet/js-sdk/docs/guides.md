# Guides

Welcome to the practical guides for the `@blocklet/js-sdk`. This section is designed to get you up and running with common tasks quickly. Each guide provides step-by-step instructions and code examples to help you integrate key features like making secure API calls, handling user authentication, and managing sessions in your Blocklet application.

These guides focus on real-world use cases to help you build robust features with confidence.
