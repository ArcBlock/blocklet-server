# APIリファレンス

`@blocklet/js-sdk`のAPIリファレンスへようこそ。このセクションでは、SDKが公開するすべてのクラス、メソッド、関数、および型に関する詳細な技術ドキュメントを提供します。SDKのコンポーネントとその機能を正確に理解する必要がある開発者を対象としています。

カスタムAPIクライアントの作成、特定のBlockletサービスとの対話、またはデータ構造の理解を目指している場合でも、必要な詳細がここにあります。

<x-cards data-columns="3">
  <x-card data-title="SDKクライアント" data-icon="lucide:box" data-href="/api/client">
    メインの`BlockletSDK`クラスと、`getBlockletSDK`、`createAxios`、`createFetch`のようなファクトリ関数のリファレンスです。
  </x-card>
  <x-card data-title="サービス" data-icon="lucide:server" data-href="/api/services">
    認証やユーザーセッションなど、特定のBlocklet APIエンドポイントと対話するサービスの詳細なAPIドキュメントです。
  </x-card>
  <x-card data-title="型" data-icon="lucide:file-code-2" data-href="/api/types">
    エクスポートされたすべてのTypeScriptの型とインターフェースの包括的なリファレンスで、アプリケーションの型安全性を確保するのに役立ちます。
  </x-card>
</x-cards>

このリファレンスは、必要な情報を素早く見つけられるように構成されています。より実践的でタスクベースの導入については、まず[ガイド](./guides.md)から始めることをお勧めします。詳細に踏み込む準備ができている場合は、上のカテゴリを選択して開始してください。
